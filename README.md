# Dhuvarrish — Full Stack Developer

Personal portfolio frontend built with Next.js 16, TypeScript, and shadcn/ui. Includes a projects page that demonstrates full stack development, backed by a companion REST API (portfolio-service (https://github.com/Dhuvarrish/portfolio-service)).

## Tech Stack

- **Framework** — Next.js 16 (App Router)
- **Language** — TypeScript
- **Styling** — Tailwind CSS, shadcn/ui, Radix UI

## Pages

| Route                            | Description                                                            |
| -------------------------------- | ---------------------------------------------------------------------- |
| `/`                              | About — about me, expertise, work approach, education & certifications |
| `/github`                        | GitHub profile, recent repos, and contribution heatmap                 |
| `/projects/backend-api-showcase` | Live demo of the backend REST API                                      |
| `/projects/microservice`         | Microservice project showcase                                          |
| `/contact`                       | Contact form                                                           |
